The global message is a **banner that appears at the very top of your website** on every page. Every single visitor will see it â€” whether they are on the homepage, browsing products, or checking out.

## Setting Up Your Message

### Step 1: Type Your Message

1. Find the **Message** text box on the page
2. Click inside the text box
3. Type the message you want to display

Keep it short and clear â€” this is a banner, not a blog post. One or two sentences is ideal.

**Good examples:**
- `Welcome to Alphasuits â€” Premium suits tailored to perfection`
- `Free delivery on all orders above Rs 3,000`
- `New winter collection now available! Shop the latest styles`
- `ðŸŽ‰ Festival Sale â€” Up to 40% off on selected items`

**Avoid:**
- Very long messages that take up too much space on the screen
- Messages that are not relevant or useful to visitors

> **Tip:** If you want to hide the message completely, just leave the text box **empty**. An empty message means no banner will be shown on the website.

### Step 2: Set an Expiry Date (Optional)

Below the message text box, you will see an **Expiry Date (Optional)** field.

This lets you set a date after which the message will automatically disappear from your website. This is very useful for temporary announcements like sales, holidays, or limited-time offers.

**How to set an expiry date:**

1. Click on the date field
2. A calendar will appear
3. Pick the date you want the message to stop showing
4. The message will automatically hide after that date â€” you do not need to come back and remove it manually

**If you do not want the message to expire:**

Leave the expiry date field empty. The message will stay visible until you manually change or remove it.

> **Tip:** Always set an expiry date for time-limited announcements like sales or holiday closures. This way, you do not have to remember to come back and remove the message â€” it happens automatically.

### Step 3: Save Your Changes

1. Click the **Save Changes** button in the top-right corner
2. Wait a moment for the changes to save
3. Open your website in a new tab to see your message at the top

> **Important:** Your message will NOT appear on the website until you click **Save Changes**. If you leave the page without saving, nothing will change.

## Editing Your Message

To change the message that is currently showing:

1. Go to **Settings â†’ Messages**
2. Click inside the Message text box
3. Delete the old text and type your new message
4. Update the expiry date if needed
5. Click **Save Changes**

The website will immediately show the updated message.

---

## Removing Your Message

To stop showing any message on your website:

1. Go to **Settings â†’ Messages**
2. Click inside the Message text box
3. Delete all the text so the box is completely empty
4. Click **Save Changes**

With an empty message, no banner will appear on your website.

## Tips for Writing Good Messages

> **Tip:** Keep it under 15 words if possible. The shorter the message, the more likely visitors will actually read it.

> **Tip:** Use emojis sparingly to make your message stand out. A single ðŸŽ‰ or âœ¨ at the start can catch the eye without looking cluttered.

> **Tip:** Change your message regularly to keep things fresh. A message that has been up for months becomes invisible â€” visitors stop noticing it.

> **Tip:** Match your message to what is happening right now. If you are running a sale, announce it. If you just restocked popular items, let people know. If nothing special is happening, a simple welcome message works fine.

> **Tip:** Use the expiry date for all temporary messages. It is easy to forget to remove a "Weekend Sale!" message on Monday morning â€” the expiry date handles it for you.

> **Tip:** Do not use the message banner for information that is already on your homepage (like your slider images). Use it for quick, important updates that visitors need to see on every page.

## Common Questions

### The message is not showing on my website. What is wrong?

Make sure you:
1. Typed something in the Message text box (an empty message will not show)
2. Clicked **Save Changes**
3. The expiry date has not already passed â€” if it has, the message has automatically hidden itself

### Can I show different messages on different pages?

No. The global message appears on **every page** of your website. If you need page-specific content, consider using the Events or Slider features instead.

### How do I add a link in the message?

The message is plain text only â€” you cannot add clickable links inside it. If you want to direct customers to a specific page, mention it in the text (e.g., "Visit our Sale page for 20% off!") and use the Slider or Events to create clickable links.

### Can I show multiple messages at the same time?

No. There is only one global message slot. If you need to communicate multiple things, combine them into one short message or prioritize the most important one.
